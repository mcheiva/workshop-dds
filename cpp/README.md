# You have chosen the C++ way! I salute you

Brief overview:
- Build Publisher/Subscriber using eProsima Fast DDS through our DDSBus API.
- Use locally generated IDL types
- Configure QoS in code or via XML profiles.

Prerequisites:
- Windows with Visual Studio, CMake, and Conan.
